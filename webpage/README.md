
# My Internship Webpage

## About
This is a simple internship webpage built using HTML, CSS, and JavaScript.  
It has a nice background, interactive buttons, and smooth animations.

## Features
- Sticky navigation bar  
- Animated hero section with heading and buttons  
- Glass-style sections for About, Features, and Contact  
- Feature cards with scroll animations  
- Footer with contact info  

## Technologies Used
- HTML  
- CSS  
- JavaScript  
- GSAP (for animations)  

## Project Files

index.html
style.css
script.js
animations.js
README.md


## How to Open
1. Download or clone this project.  
2. Open index.html in your web browser.  
3. Click buttons and scroll to see animations.

## Purpose
This project is created for learning web development and as part of an internship task.
